
# Awaken (Academic Work)
Awaken is a Hack 'n' Slash game made with Phaser

![Awaken](http://awaken.ch/game.png)

## How to play
### Online
- Open [Demo](http://awaken.ch)

### Offline
- Run `php -S localhost:8080` in `/`
- Browse `localhost:8080`
